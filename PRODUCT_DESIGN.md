# ResumeAI - 智能简历助手产品方案

## 一、产品定位

**ResumeAI** 是一款本地优先的智能简历助手，帮助用户：
1. 建立个人技能/经历知识库
2. 根据岗位 JD 自动生成针对性简历
3. 分析技能缺口，推荐学习路径

---

## 二、核心价值

| 用户痛点 | 解决方案 |
|----------|----------|
| 每次投简历都要重新写 | 个人数据库 + AI 自动生成 |
| 不知道简历该怎么写 | AI 根据JD优化内容 |
| 不知道自己缺什么技能 | 技能分析 + 学习推荐 |
| 担心隐私泄露 | 本地存储 + 本地 API Key |

### 4.3 JD 解析与简历生成

#### 工作流程
```
输入 JD → AI 提取关键信息 → 匹配个人数据库 → 生成简历 → 用户微调
```

#### JD 解析
```json
{
  "parsedJD": {
    "position": "高级前端工程师",
    "company": "某互联网公司",
    "requiredSkills": ["React", "TypeScript", "Node.js"],
    "preferredSkills": ["Next.js", "GraphQL"],
    "responsibilities": [
      "负责前端架构设计",
      "优化页面性能"
    ],
    "requirements": [
      "3年以上经验",
      "有大型项目经验"
    ],
    "keywords": ["性能优化", "架构", "团队协作"]
  }
}
```

#### 简历生成策略
1. **技能匹配**：优先展示 JD 要求的技能
2. **经历筛选**：选择与 JD 最相关的经历
3. **内容优化**：AI 改写经历描述，突出关键词
4. **项目推荐**：选择能体现相关技能的项目

## 五、用户界面设计

```

├── 数据概览
│   ├── 技能统计（雷达图）

    ├── 学习推荐
    └── 技能更新

数据管理
├── 技能库
├── 经历库
├── 项目库

├── 技能缺口分析
├── 学习路径推荐
├── 开源项目推荐
└── 学习进度追踪

设置
├── AI 模型配置
├── API Key 管理
├── 导入/导出数据
└── 主题设置
```

### 5.2 核心交互流程

#### 流程1：根据 JD 生成简历
```
1. 用户粘贴 JD
2. AI 解析 JD → 提取关键词
3. 系统匹配个人数据库
4. AI 生成针对性简历
5. 用户预览、微调
6. 导出 PDF/JSON
```

#### 流程2：技能分析
```
1. 用户选择目标岗位类型
2. 系统分析行业技能需求
3. 对比个人技能库
4. 输出技能缺口报告
5. 推荐学习路径
```

## 六、技术方案

### 6.1 技术栈

| 层级 | 技术选型 |
|------|----------|
| 前端 | React + TypeScript + TailwindCSS |
| 状态管理 | Zustand |
| 本地存储 | IndexedDB + localStorage |
| AI 接口 | 统一适配层 |
| 构建 | Vite |
| 部署 | 本地应用 / Web 应用 |

### 6.2 数据流

```
用户输入 → 本地存储 → AI 处理 → 结果展示
                ↓
            加密备份（可选）
```

### 6.3 安全设计

1. **API Key 安全**
   - 本地加密存储
   - 不上传到任何服务器
   - 使用时才解密

2. **数据隐私**
   - 默认本地存储
   - AI 调用只发送必要数据
   - 用户可选择脱敏
